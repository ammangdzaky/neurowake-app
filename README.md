# 🧠 NeuroWake v1.0

**NeuroWake** adalah aplikasi alarm Android cerdas yang dirancang untuk memastikan Anda benar-benar bangun dengan kondisi otak yang aktif. Menggunakan konsep "Neuro-Logic", aplikasi ini memaksa Anda menyelesaikan tantangan matematika sebelum alarm dapat dimatikan.

---

## 🚀 Fitur Unggulan

*   **⚡ Neuro-Logic Challenge**: Pilih tingkat kesulitan (Easy/Hard) untuk tantangan matematika yang harus diselesaikan agar alarm berhenti. 
*   **🎵 Smart Audio Library**: Impor file audio lokal (MP3/FLAC) Anda sendiri. Aplikasi akan mengacak lagu yang Anda pilih setiap kali alarm berbunyi agar Anda tidak bosan.
*   **📊 Discipline Stats & Ranking**: Pantau kedisiplinan bangun pagi Anda. Semakin sering Anda berhasil, semakin tinggi "Discipline Rank" Anda (dari Newbie hingga Master).
*   **🔔 Upcoming Alarm Notification**: Notifikasi cerdas 30 menit sebelum alarm berbunyi, memberikan opsi untuk "Skip" jika Anda sudah bangun lebih awal.
*   **🔐 Lockscreen Bypass**: Alarm akan otomatis menyalakan layar dan tampil di atas layar kunci, memastikan Anda tidak melewatkan tantangan.
*   **👤 Personal Profile**: Kustomisasi profil dengan nama, foto, dan dynamic badge yang menunjukkan level kedisiplinan Anda.
*   **🎨 Modern Dark UI**: Desain antarmuka yang elegan dengan tema gelap dan aksen Cyan yang futuristik.

## 📥 Cara Download & Instalasi

Untuk mencoba aplikasi ini di perangkat Android Anda, ikuti langkah berikut:

1.  Buka halaman **[Releases](https://github.com/ammangdzaky/NeuroWake/releases)** di repositori ini.
2.  Download file **`NeuroWake-v1.0.apk`**.
3.  Buka file APK tersebut di HP Anda.
4.  Jika muncul peringatan keamanan, aktifkan izin **"Install from Unknown Sources"** di pengaturan HP Anda.
5.  Selesaikan instalasi dan buka aplikasi.

## 🛠️ Teknologi yang Digunakan

*   **Language**: Java
*   **UI Framework**: Material Design Components (Google)
*   **Storage**: Internal Storage (File I/O) & SharedPreferences
*   **Android API**: AlarmManager, NotificationManager, Foreground Service, KeyguardManager.
